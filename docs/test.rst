Running tests
=============

KoNLPy has tests to test its quality.
To perform a test, enter the code below.

.. code-block:: bash

    $ pip install pytest
    $ cd konlpy
    $ py.test

.. note::

    KoNLPy was tested on the below environments:

    - Ubuntu 12.04 with openjdk-7-jdk
    - Ubuntu 13.10 with openjdk-7-jdk
